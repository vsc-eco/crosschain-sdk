Magi QuickSwap — SDK demo

Embedded @vsc.eco/crosschain-widget driving @vsc.eco/crosschain-sdk against mainnet. Same scope as Altera's QuickSwap card: HIVE/HBD → HIVE/HBD/BTC.

What this demo proves
  • Widget renders and mounts in a plain HTML page (zero framework imports beyond React).
  • Preview math updates live against real pool depths from api.vsc.eco.
  • Ops are built via @vsc.eco/crosschain-sdk — same JSON shape Altera broadcasts.
  • Signing handoff to Aioha (Keychain / HiveSigner / HiveAuth / PeakVault / Ledger) is wired.
